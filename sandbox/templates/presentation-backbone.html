What is Backbone?

  • a Javascript framework that allows you to structure your Javascript code in an MVC (Model, View, Controller) fashion
  • Only dependency is underscore.js "a utility-belt library for JavaScript that provides a lot of the functional programming support that you would expect in Prototype.js (or Ruby), but without extending any of the built-in JavaScript objects"
  • Jquery not required, but usually used for DOM manipulation (not using Jquery for everything is the new using Jquery for everything)

Why would you use it?

  • Add structure and process to your JavaScript application
  • Some really nice features built in to standardise the way common tasks are carried out
  • Really useful way of making an application event driven, rather than relying on scattering listeners around the DOM and chaining functions together as I have been doing up until now
  • Models
  • Views
  • Routers
  • Collections
  • Ajax/ REST
